Admin Console

Facebook / Instagram data-deletion grace period

When a Meta account is disconnected (in-app or removed on Facebook's side), its data is hidden immediately and permanently deleted after this window. Reconnecting before then cancels the deletion.

Changing this affects only deletions started from now on — requests already counting down keep their original date. Default is 30.

Global toggles

Platform-wide switches. Turning one off overrides the corresponding per-tenant setting everywhere.

Maintenance modePuts the portal in read-only and shows a banner to every tenant.
Allow new sign-upsWhen off, the public registration flow is closed.
Outbound SMS enabledMaster kill-switch for all Twilio sends across the platform.
Enforce webhook signature verificationSquare, Twilio & Shopify signatures required (v1 had these disabled).
