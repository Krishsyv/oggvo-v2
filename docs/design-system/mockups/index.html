OGGVO v2 — Mock UI

Browser-openable mockups for every product domain, built on the v2 design system and the reusable OggvoShell. All screens are interactive — filters, tabs, modals, drag-reorder, and live previews work (sample data, no app code). Toggle dark mode top-right.

Docs: design system · contacts · reviews+funnel · campaigns · messaging · social · surveys · widgets · analytics · integrations · compliance · settings · admin · auth
