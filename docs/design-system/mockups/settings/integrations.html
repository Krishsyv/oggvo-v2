Disconnect Provider?

Disconnecting Provider revokes its access & refresh tokens with the provider and tears down its registered webhooks.

Child data harvested through this connection (reviews, posts, insights, messages, imported contacts) is soft-deleted as a set and hard-purged after a 30-day grace window. Reconnecting within the window restores it.
Meta Data Deletion Callback: this also records a deletion_request (30-day grace) so Facebook's Data Deletion Request callback is satisfied automatically — no manual purge emails.
